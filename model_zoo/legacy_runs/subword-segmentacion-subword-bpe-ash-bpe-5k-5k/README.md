### Modelos heredados de EIB Spellchecker

#### Proposito

Esta carpeta contiene artefactos de modelos, configuraciones, salidas o recursos generados durante corridas heredadas del proyecto EIB Spellchecker.

#### Uso

Estos archivos sirven como referencia para reproducibilidad, comparacion experimental o migracion hacia versiones mas organizadas del corrector.

#### Nota

No modificar pesos, checkpoints ni artefactos binarios manualmente. Cualquier actualizacion debe quedar documentada mediante commits y reportes reproducibles.
